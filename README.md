# Personal Portfolio Website

An ultra-performant, accessible single-page portfolio website built with modern web technologies and best practices.

## Features

### 🚀 Performance Optimizations
- **Critical CSS inlined** for faster initial render
- **Lazy loading** for images and off-screen content
- **Intersection Observer** for efficient scroll animations
- **Debounced scroll events** to prevent performance issues
- **Resource preloading** for improved navigation speed
- **Minification-ready** CSS and JavaScript structure
- **WebP support** detection and optimization

### ♿ Accessibility Features
- **WCAG 2.1 AA compliant** design
- **Skip links** for keyboard navigation
- **ARIA labels** and semantic HTML5 structure
- **Screen reader announcements** for dynamic content
- **Keyboard navigation** with focus management
- **High contrast mode** toggle
- **Large font mode** for better readability
- **Focus indicators** and keyboard traps
- **Reduced motion** support

### 🎨 Design & UX
- **Responsive design** for all devices
- **Dark/Light/Auto theme** switching
- **Smooth animations** with hardware acceleration
- **Mobile-first** approach
- **Modern CSS Grid** and Flexbox layouts
- **Custom CSS properties** for theming
- **Print-friendly** styles

### 📱 Sections
- **Hero** - Introduction with call-to-action
- **Experience** - Timeline of professional experience
- **Diplomas** - Education and certifications
- **Projects** - Featured work showcase
- **Extras** - Additional activities and interests
- **Contact** - Contact form and information

## Technologies Used

- **HTML5** - Semantic markup and accessibility
- **CSS3** - Modern features with custom properties
- **Vanilla JavaScript** - No framework dependencies
- **Web APIs** - Intersection Observer, Performance Observer
- **CSS Grid & Flexbox** - Responsive layouts

## Performance Metrics

The website is optimized for:
- **First Contentful Paint (FCP)**: < 1.5s
- **Largest Contentful Paint (LCP)**: < 2.5s
- **First Input Delay (FID)**: < 100ms
- **Cumulative Layout Shift (CLS)**: < 0.1

## Accessibility Compliance

- **WCAG 2.1 AA** compliant
- **Screen reader** friendly
- **Keyboard navigable**
- **Color contrast** ratios meet standards
- **Focus management** for interactive elements

## Customization

### Personal Information
Update the following in `index.html`:
- François Eoche in the hero section and navigation
- Contact information in the contact section
- Experience timeline entries
- Education and certifications
- Project details and links

### Styling
Modify `styles.css` to customize:
- Color scheme (CSS custom properties)
- Typography and fonts
- Spacing and layout
- Animation timing

### Functionality
Extend `script.js` to add:
- New interactive features
- Additional form validation
- Custom animations
- Analytics integration
